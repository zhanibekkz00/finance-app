# Как запустить приложение на телефоне

## Способ 1: Физический Android телефон (рекомендуется)

### Шаг 1: Подготовка телефона

1. **Включите режим разработчика:**
   - Откройте "Настройки" → "О телефоне"
   - Найдите "Номер сборки" и нажмите на него 7 раз
   - Появится сообщение "Вы стали разработчиком!"

2. **Включите отладку по USB:**
   - Откройте "Настройки" → "Для разработчиков"
   - Включите "Отладка по USB"
   - (Опционально) Включите "Оставаться активным при зарядке"

3. **Подключите телефон к компьютеру:**
   - Используйте USB кабель
   - На телефоне появится запрос "Разрешить отладку по USB?" → нажмите "Разрешить"
   - Отметьте "Всегда разрешать с этого компьютера" (если хотите)

### Шаг 2: Проверка подключения

Откройте PowerShell в папке проекта и выполните:
```powershell
.\flutter\bin\flutter.bat devices
```

**Важно:** Используйте `.\flutter\bin\flutter.bat` вместо просто `flutter`, так как Flutter не добавлен в PATH.

Вы должны увидеть ваш телефон в списке, например:
```
sdk gphone64 arm64 • emulator-5554 • android-arm64 • Android 14
```

### Шаг 3: Авторизация устройства

Если видите сообщение "Device ... is not authorized":
1. **На телефоне** появится диалог "Разрешить отладку по USB?"
2. Нажмите **"Разрешить"**
3. Отметьте **"Всегда разрешать с этого компьютера"**
4. Повторно выполните: `.\flutter\bin\flutter.bat devices`

### Шаг 4: Запуск приложения

```powershell
.\flutter\bin\flutter.bat run
```

Или укажите конкретное устройство:
```powershell
.\flutter\bin\flutter.bat run -d <device-id>
```

## Способ 2: Android эмулятор

### Шаг 1: Создание эмулятора

1. Откройте **Android Studio**
2. Перейдите в **Tools** → **Device Manager**
3. Нажмите **Create Device**
4. Выберите устройство (например, Pixel 5)
5. Выберите системный образ (рекомендуется API 33 или выше)
6. Нажмите **Finish**

### Шаг 2: Запуск эмулятора

1. В Android Studio: **Device Manager** → выберите эмулятор → нажмите ▶️
2. Или через командную строку:
```powershell
.\flutter\bin\flutter.bat emulators --launch <emulator-id>
```

### Шаг 3: Запуск приложения

```powershell
.\flutter\bin\flutter.bat run
```

## Способ 3: Через ADB (если телефон не определяется)

### Установка ADB

1. Скачайте [Android Platform Tools](https://developer.android.com/studio/releases/platform-tools)
2. Распакуйте в папку (например, `C:\adb`)
3. Добавьте в PATH: `C:\adb`

### Проверка подключения

```powershell
adb devices
```

**Примечание:** Если `adb` не найден, используйте полный путь или добавьте в PATH.

Должен показать:
```
List of devices attached
ABC123XYZ    device
```

Если показывает `unauthorized`, на телефоне разрешите отладку по USB.

## Решение проблем

### Проблема: "No devices found"

**Решение:**
1. Проверьте, что USB отладка включена на телефоне
2. Попробуйте другой USB кабель
3. Установите драйверы для вашего телефона (обычно автоматически через Windows Update)
4. Перезапустите ADB: `adb kill-server` затем `adb start-server`

### Проблема: "Android license status unknown"

**Решение:**
```powershell
.\flutter\bin\flutter.bat doctor --android-licenses
```
Примите все лицензии, нажимая `y` (yes).

### Проблема: "cmdline-tools component is missing"

**Решение:**
1. Установите [Android Studio](https://developer.android.com/studio)
2. Или установите только [Command Line Tools](https://developer.android.com/studio#command-line-tools-only)
3. Установите переменную окружения `ANDROID_HOME`

## Для iOS (Mac только)

Если у вас Mac и iPhone:

1. Подключите iPhone через USB
2. Доверьтесь компьютеру на iPhone
3. Включите режим разработчика на iPhone
4. Запустите: `flutter run`

**Примечание:** Для iOS нужен Mac и установленный Xcode.

---

## Быстрая проверка

После подключения телефона выполните в PowerShell:

```powershell
# Проверить устройства
.\flutter\bin\flutter.bat devices

# Запустить приложение
.\flutter\bin\flutter.bat run

# Или с горячей перезагрузкой
.\flutter\bin\flutter.bat run --hot
```

## Важно: Правильные команды для PowerShell

Так как Flutter не добавлен в PATH, всегда используйте:
- `.\flutter\bin\flutter.bat` вместо `flutter`
- Или добавьте Flutter в PATH для удобства

Приложение установится на телефон и автоматически откроется!
