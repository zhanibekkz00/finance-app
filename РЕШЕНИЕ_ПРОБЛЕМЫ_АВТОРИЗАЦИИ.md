# Решение проблемы авторизации телефона

## Проблема: "Device ... is not authorized"

Ваш телефон подключен, но не авторизован для отладки.

## Решение (пошагово):

### Шаг 1: Проверьте телефон

1. **Разблокируйте телефон** (экран должен быть включен)
2. **Проверьте, появился ли диалог** "Разрешить отладку по USB?"
   - Если появился → нажмите **"Разрешить"**
   - Отметьте **"Всегда разрешать с этого компьютера"**
   - Нажмите **"OK"**

### Шаг 2: Если диалог не появился

1. **Отключите и снова подключите USB кабель**
2. **На телефоне:**
   - Откройте **Настройки** → **Для разработчиков**
   - Выключите и снова включите **"Отладка по USB"**
   - Попробуйте отключить и включить **"USB-накопитель"** или **"Передача файлов"**

### Шаг 3: Проверьте режим USB

На телефоне при подключении выберите:
- **"Передача файлов"** (File Transfer / MTP)
- Или **"Передача файлов (MTP)"**
- **НЕ** используйте "Только зарядка"

### Шаг 4: Перезапустите проверку

После выполнения шагов выше, выполните:

```powershell
.\flutter\bin\flutter.bat devices
```

Должно появиться:
```
PNM7UOXKS8J7IVUO    device    (вместо "unauthorized")
```

### Шаг 5: Если все еще не работает

1. **Отключите телефон от USB**
2. **На телефоне:** Настройки → Для разработчиков → **"Отменить авторизацию USB"**
3. **Подключите телефон снова**
4. **Разрешите отладку** когда появится диалог
5. **Проверьте снова:** `.\flutter\bin\flutter.bat devices`

## Альтернативное решение: Через Android Studio

1. Откройте **Android Studio**
2. Подключите телефон
3. Android Studio автоматически попросит разрешение
4. После авторизации в Android Studio, Flutter тоже увидит устройство

## После успешной авторизации

Когда увидите `device` (не `unauthorized`), запустите приложение:

```powershell
.\flutter\bin\flutter.bat run
```

Приложение установится и откроется на вашем телефоне!
